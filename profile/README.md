## 🧑‍💻 Tech Stack
<img width="2064" alt="stack" src="https://github.com/98OO/.github/assets/70826982/65bad794-ad3a-48ff-a489-867f7cef5aaa">

### Frontend
- **Language**
    - 100% Typescript

- **라이브러리 및 프레임워크**
    - React, TanStack Query, Styled components
      
- **테스트**
    - Storybook, MSW
 
### Backend
- **Language**
    - 100% JAVA

- **라이브러리 및 프레임워크**
    - Spring Boot, Spring Data Jpa, Gradle, Lombok
      
- **데이터베이스**
    - MySQL, Redis
   
- **테스트**
    - JUnit5, Swagger

### 공통
- **형상 관리**
    - Github

- **인프라**
    - Amazon Web Service, Cloudflare, Docker Compose, Github Actions,
 
- **협업 도구**
    - Notion, Discord, Figma
